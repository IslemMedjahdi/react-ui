# TextLink

This component is used to link to an internal or external page.

It leverages on the LinkComponent in which a specific router for a project can be defined e.g. next/router.

export const Template = ({ href, children, ...args }) => (
  <TextLink {...args} href={href}>
    {children}
  </TextLink>
)

## Theme

- `textLink.base`: Define the underline and focus interaction.
- `textLink.normal`: Define the text centering for the component.
