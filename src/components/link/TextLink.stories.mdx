# TextLink

This component is used to link to an internal page.

It leverages on the LinkComponent in which a specific router for a project can be defined e.d. next/router.

export const Template = ({ href, children, ...args }) => (
  <TextLink {...args} href={href}>
    {children}
  </TextLink>
)
