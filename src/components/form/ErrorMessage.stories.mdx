### Theme

- `bg-error-message`: Define the default background of the error message.
- `text-error-message-content`: Define the default color of the error message content.
- `text-error-message-icon`: Define the default color of the error message icon.
- `bg-error-message-icon`: Define the default background color of the error message icon.
