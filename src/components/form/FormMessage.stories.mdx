# FormMessage

This component displays messages within a form with an icon and text.
It supports different types of tones.

export const Template = ({ children, ...args }) => (
  <FormMessage {...args}>{children}</FormMessage>
)

<ArgsTable story="Default" />

### Theme

- `border-form-message-${tone}`: Define the default border color of the form message per tone.
- `bg-error-message-${tone}`: Define the default background of the form message per tone.
- `text-error-message-content-${tone}`: Define the default color of the form message content per tone.
- `text-error-message-icon-${tone}`: Define the default color of the form message icon per tone.
- `bg-error-message-icon-${tone}`: Define the default background color of the form message icon per tone.
