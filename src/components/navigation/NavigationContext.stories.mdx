# NavigationProvider

This component keeps track of state of the navigation toggle using React Content API.
The [HeaderArea](/docs/components-header-headerarea--default-story) expects to be nested inside a NavigationProvider.

export const Template = ({ ...args }) => (
  <NavigationProvider {...args}>
    <HeaderArea>
      <HeaderTitle>Header Title</HeaderTitle>
    </HeaderArea>
  </NavigationProvider>
)
