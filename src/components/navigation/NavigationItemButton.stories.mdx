# NavigationItemButton

This button can be placed inside the navigation. E.g. Logout

export const Template = ({ ...args }) => (
  <NavigationItemButton {...args} content="Log out" icon={IconLogout} />
)

<Canvas>
  <Story name="Default">{Template.bind({})}</Story>

### Props

<ArgsTable story="Default" />

### Theme

- `navigation.item.base`: Define the base style of the button, that is shared between different states.
- `navigation.item.normal`: Define the normal style of the button.
- `navigation.item.disabled`: Define the disabled style of the button.
- `navigation.item.icon.base`: Define the size of the icon.
- `navigation.item.content.base`: Define the floating direction of the content.
