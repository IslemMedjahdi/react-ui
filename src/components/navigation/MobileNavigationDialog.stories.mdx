This component is used to provide a mobile menu.

It leverages on the component [MobileNavigationDialogCloseButton](/docs/components-menu-MobileNavigationDialogCloseButton--default-story)

_**Important**_: this storybook is only available for a mobile view. Go to Canvas -> change the size of the preview -> Small mobile.
