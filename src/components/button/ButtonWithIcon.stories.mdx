# ButtonWithIcon

The **_default values_** for variant, size, and tone can be found in the [Button Section](/docs/components-button-button--default-story).

The size of the icon is determined by the `size` prop of the component using the height and width Tailwind classes to ensure proper scaling of the icon. `Size.md` will result in `w-6 h-6` to be passed to the icon component, while for `Size.sm` the classes applied are `w-4 h-4`.

## Usage

```jsx
<ButtonWithIcon Icon={Icon} onClick={() => console.log('Clicked')}>
  Button With Icon
</ButtonWithIcon>
```

### Theme

- `button.withIcon.iconContainer.base`: Define the positioning of the icon of button with icon.
- `button.withIcon.icon.base`
- `button.withIcon.icon.size.sm`: Define the size of the icon of button with icon.
- `button.withIcon.icon.size.md`: Define the size of the icon of button with icon.
