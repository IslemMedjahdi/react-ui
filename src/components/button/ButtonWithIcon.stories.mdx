# ButtonWithIcon

The **_default values_** for variant, size, and tone can be found in the <LinkTo kind="components-button-button--default-story">Button Section</LinkTo>.

The size of the icon is determined by the `size` prop of the component using the height and width Tailwind classes to ensure proper scaling of the icon. `Size.md` will result in `w-4 h-4` to be passed to the icon component, while for `Size.sm` the classes applied are `w-3 h-3`.
