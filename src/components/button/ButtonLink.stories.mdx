# ButtonLink

The ButtonLink component has the looks of a Button, but is an anchor tag under the hood.
You can define the type of link in the DesignSystemProvider. By default, it will be a plain HTML anchor tag.

export const Template = ({ children, ...args }) => (
  <ButtonLink {...args}>{children}</ButtonLink>
)

### Theme

- `button.buttonLink.base`
- `button.buttonLink.size.sm`: Define the size of the small size button.
- `button.buttonLink.size.md`: Define the size of the medium size button.
- `button.buttonLink.variantTone.solid.base`
- `button.buttonLink.variantTone.solid.primary`: Define the color and interactions for the button with the variant solid and tone primary.
- `button.buttonLink.variantTone.solid.critical`: Define the color and interactions for the button with the variant solid and tone critical.
- `button.buttonLink.variantTone.solid.secondary`: Define the color and interactions for the button with the variant solid and tone secondary.
- `button.buttonLink.variantTone.solid.disabled`: Define the status disabled for the button with the variant solid.
- `button.buttonLink.variantTone.ghost.base`
- `button.buttonLink.variantTone.ghost.primary`: Define the color and interactions for the button with the variant ghost and tone primary.
- `button.buttonLink.variantTone.ghost.critical`: Define the color and interactions for the button with the variant ghost and tone critical.
- `button.buttonLink.variantTone.ghost.secondary`: Define the color and interactions for the button with the variant ghost and tone secondary.
- `button.buttonLink.variantTone.ghost.disabled`: Define the status disabled for the button with the variant ghost.
- `button.buttonLink.variantTone.transparent.base`
- `button.buttonLink.variantTone.transparent.primary`: Define the color and interactions for the button with the variant transparent and tone primary.
- `button.buttonLink.variantTone.transparent.critical`: Define the color and interactions for the button with the variant transparent and tone critical.

- `button.buttonLink.variantTone.transparent.disabled`: Define the status disabled for the button with the variant transparent.
