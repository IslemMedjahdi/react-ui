# LanguagePicker

This component is used to set the default language of the browser and provides the possibility to switch to another given language.

_**Important**_: the text and icon color for the tailwind class `lg` are set to white.

export const Template = ({ children, menuLabel, className, ...args }) => (
  <LanguagePicker {...args} menuLabel={menuLabel} className={className}>
    {children}
  </LanguagePicker>
)

### Theme

- `languagePicker.menuButton.base`: Define the flexbox and the interactions with hover and active for the menu button.
- `languagePicker.menuButton.text.base`
- `languagePicker.menuButton.text.normal`: Define the text color for the language to the left of the icon.
- `languagePicker.menuButton.icon.base`: Define the width and height for the icon.
- `languagePicker.menuButton.icon.normal`: Define the color for the icon.
- `languagePicker.menuList.base`: Define the margin and padding for the menu list.
- `languagePicker.menuList.normal`: Define the background color of the menu list.
- `languagePicker.menuList.menuLink.base`: Define the padding and interaction with hover for the menu item.
