# SectionListItemError

This component can be used to show the user that there was an error fetching data.

It is often used on a page that consists of multiple sections. In such cases you often don't want to show the full [SectionError](/docs/components-section-sectionerror--default-story).

The component is base on the [SectionListItem](/docs/components-section-sectionlistitem--default-story).

export const Template = ({ children }) => (

)

### Theme

- `section.contentList.error.base`: Define the base style of them error section content list.
- `section.contentList.error.circle.base`: Define the color of the circle in which the icon is placed.
- `section.contentList.error.icon.base`: Define the padding of the icon.
- `section.contentList.error.icon.normal`: Define the color, and the shape of the icon.
- `section.contentList.error.children.base`: Define the text color of the passed children.
