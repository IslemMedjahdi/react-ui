# SectionContentError

This component can be used to show the user that there was an error fetching data.
It wraps a warning icon with a custom message within a [SectionContent](/docs/components-section-sectioncontent--default-story)

It is often used on a page that consists of multiple sections. In such cases you often don't want to show the full [ContentError](/docs/components-section-sectionerror--default-story).

export const Template = ({ children }) => (
  <SectionContentError>{children}</SectionContentError>
)

### Theme

- `section.content.error.container.base`: Define the styles of the error section content.
- `section.content.error.iconContainer.base`: Define the styles of the icon container in which the icon is placed.
- `section.content.error.icon.base`: Define the styles of the icon.
- `section.content.error.children.base`: Define the styles of the passed children.
