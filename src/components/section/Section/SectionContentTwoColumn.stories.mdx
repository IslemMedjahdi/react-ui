# SectionContentTwoColumn

This component builds on top of the `SectionContent` component and applies a grid styling to it.
On screen sizes below `xl` it will have one column and on larger screens two.

export const Template = ({ children, ...args }) => (
  <SectionContentTwoColumn {...args}>{children}</SectionContentTwoColumn>
)
