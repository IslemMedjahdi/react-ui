### Theme

- `section.header.base`: Define the styles for the section heading.

## Theme Overwrite

It is possible to overwrite the styles. In the following example, it is moderated what needs to be passed to the function makeTheme.

```js
const themeOverwrite = {
  section: {
    header: {
      base: 'p-5 lg:p-10',
    },
  },
}
```
