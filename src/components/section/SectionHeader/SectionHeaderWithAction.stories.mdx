# SectionHeaderWithAction

This component puts the title inside the `SectionTitle` of the `SectionHeader` and pushes the action all the way to the right.
It is often used to trigger an action in the context of this section.

export const Template = ({ title, action }) => (
  <SectionHeaderWithAction title={title} action={action} />
)
