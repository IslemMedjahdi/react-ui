# SectionListItemWithAction

The SectionListItemWithAction defines a row in a list of items in a section and pushes an action to right of the content.

export const Template = ({ children, action }) => (
  <SectionListItemWithAction action={action}>
    {children}
  </SectionListItemWithAction>
)
