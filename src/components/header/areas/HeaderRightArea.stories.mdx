# HeaderRightArea

This component is mostly used in the header and represents the children on the right side of the header. For example it is used to show the search button.
It applies a margin to the left and displays the children in a row.

export const Template = ({ ...args }) => (
  <HeaderRightArea {...args}>
    <HeaderSmallAction
      icon={IconEdit}
      label="Edit"
      onClick={action('onClick')}
    />
  </HeaderRightArea>
)
