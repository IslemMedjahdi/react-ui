# MenuProvider

This component keeps track of state of the menu toggle using React Content API.
The [HeaderArea](/docs/components-header-headerarea--default-story) expects to be nested inside a MenuProvider.

export const Template = ({ ...args }) => (
  <IntlProvider locale="en">
    <MenuProvider {...args}>
      <HeaderArea>
        <HeaderTitle>Header Title</HeaderTitle>
      </HeaderArea>
    </MenuProvider>
  </IntlProvider>
)
