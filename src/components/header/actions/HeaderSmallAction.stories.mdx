# HeaderSmallAction

This component is used to add a _small_ icon to the header. It applies some basic styling for the icon.

**_Important:_** if the `className` is set individually the default styling will be discarded!

Default styling: `hover:text-header-action-small-hover focus:text-header-action-small-focus`

export const Template = ({ ...args }) => (
  <HeaderSmallAction
    {...args}
    icon={Icon}
    label="Enter search term"
    onClick={() => alert('Button got pressed')}
  />
)

<Canvas>
  <Story name="Default">{Template.bind({})}</Story>
</Canvas>

### Props:

- `text-header-action-hover`: Defines the color on hover
- `text-header-action-focus`: Defines the color on focus
