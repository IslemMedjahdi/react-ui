# HeaderSmallAction

This component is used to add a _small_ icon to the header. It applies some basic styling for the icon.

export const Template = ({ ...args }) => (
  <HeaderSmallAction
    {...args}
    icon={Icon}
    label="Enter search term"
    onClick={() => alert('Button got pressed')}
  />
)

<ArgsTable story="Default" />

### Theme

- `header.largeAction.normal`: Define the hover and focus interaction of the component.
- `header.largeAction.icon.base`: Define the padding of the icon.
