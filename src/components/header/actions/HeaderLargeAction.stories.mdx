# HeaderLargeAction

This component is used to add a _large_ icon to the header. It applies some basic styling for the icon.

export const Template = ({ ...args }) => (
  <HeaderLargeAction
    {...args}
    icon={Icon}
    onClick={action('onClick')}
    label="Enter search term"
  />
)

<Canvas>
  <Story name="Default">{Template.bind({})}</Story>
</Canvas>

### Props

<ArgsTable of={HeaderLargeAction} />

### Theme

- `header.largeAction.base`: Define the hover and focus interaction of the component.
- `header.largeAction.icon.base`: Define the padding of the icon.
