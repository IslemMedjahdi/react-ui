# HeaderSearch

This component contains the logic for a search functionality. It is used e.g., in the component [HeaderMainWithSearch](http://localhost:4000/?path=/docs/components-header-headermainwithsearch--default-story).

Furthermore, an adjusted interface for the mobile view is provided.

export const Template = ({ ...args }) => {
  const [text, setText] = useState('')
  const clearSearch = () => setText('')
  return (
    <NavigationProvider>
      <HeaderSearch
        {...args}
        text={text}
        setText={setText}
        clearSearch={clearSearch}
        stopSearch={clearSearch}
      />
    </NavigationProvider>
  )
}

### Theme

- `header.search.base`: Define the styles of the component.
- `header.search.input.base`: Define the styles of the input.
- `header.search.clearButton.base`: Define the styles for the button.
- `header.search.icon.base`: Define the styles of the icon.

### Internationalization

- `shared.search.placeholder`: Placeholder text for search field
- `shared.search.clear`: Accessibility label for clear button in the search field
- `shared.search.back`: Accessibility label for return button (mobile only)
