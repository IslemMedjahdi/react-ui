# HeaderBackWithEdit

This component is specifically designed for detail pages, where you want to give the user the possibility to go back to the previous page, and an edit link to update the content.

It leverages on the components [HeaderBackAction](/docs/components-header-headerbackaction--default-story), [HeaderArea](/docs/components-header-headerarea--default-story), and [HeaderEditAction](/docs/components-header-headereditaction--default-story).
