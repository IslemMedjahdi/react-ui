# HeaderBackWithEdit

This component is similar to [HeaderMainWithSearch](http://localhost:4000/?path=/docs/components-header-headermainwithsearch--default-story), but instead of the search, an edit icon is placed at the right, which will forward a user to the (edit-)page.

For the mobile view, an adjusted interface for the search is provided.
