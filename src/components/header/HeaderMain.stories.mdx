# HeaderMain

This component can be used to create swiftly a header. The Header Main should be used for pages that are on the root of a navigation stack. On mobile Main headers will have the burger menu icon on the left, whereas other Headers have a back button instead.

**_Important_**: This component needs to be wrapped in a "IntlProvider" and "MenuProvider". The detailed description of them can be found in the section of [HeaderArea](/docs/components-header-headerarea--default-story).

export const Template = ({ ...args }) => (
  <IntlProvider
    locale="en"
    messages={{
      'app.nav.menu': 'Menu',
    }}
  >
    <MenuProvider>
      <HeaderMain {...args} title="Header Title" />
    </MenuProvider>
  </IntlProvider>
)
