# Getting Started

Install the library using the following command:

```shell
npm install @aboutbits/react-ui
```

## Set up Provider

For React-UI to work properly you need to setup the `ReactUIProvider` at the root of your project:

```jsx
// 1. import `ReactUIProvider` component
import { ReactUIProvider } from '@aboutbits/react-ui'

function App({ Component }) {
  // 2. Use at the root of your app
  return (
    <ReactUIProvider>
      <Component />
    </ReactUIProvider>
  )
}
```

If you are using NextJS go to `pages/_app.js` or `pages/_app.tsx` (create one if it doesn't exist) and add this:

```jsx
import { ReactUIProvider } from '@aboutbits/react-ui'

function MyApp({ Component, pageProps }) {
  return (
    <ReactUIProvider>
      <Component {...pageProps} />
    </ReactUIProvider>
  )
}

export default MyApp
```

## Set up Tailwind

To use ReactUI with its default theme the following Tailwind configuration is necessary:

  purge: {
    content: [
      './src/**/*.js',
      './src/**/*.jsx',
      './src/**/*.ts',
      './src/**/*.tsx',
    ],
  },
  theme: {
    extend: {
      minWidth: {
        dialog: '17.5rem',
      },
      minHeight: {
        14: '3.5rem',
      },
    },
    colors: {
      white: '#FFFFFF',
      black: '#000000',
      transparent: 'rgba(0,0,0,0)',
      primary: {
        DEFAULT: '#4C6EE8',
        50: '#f6f8fe',
